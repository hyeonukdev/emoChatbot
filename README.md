# Emochatbot-chatterbot

## Flask 실행 방법
**1. Repository 가져오기**
  ```
  git clone [REPO_URL]
   ```

**2. 가상환경 생성**
  ```
  python -m venv myvenv
  ```

**3. 가상환경 실행(window)** 
  ```
  source myvenv/Scripts/activete
  ```

**4. 패키지 설치**
  ```
  pip install -r requirements.txt
  ```
**4-1. 개별 설치**

    `pip install Flask`

    `pip install chatterbot`

    `pip install chatterbot-corpus`


**5. Flask 서버 실행**
  ```
  python app.py
  ```
